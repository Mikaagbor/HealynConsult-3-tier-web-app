# HealynConsult

A Dockerized 3-Tier Web App (Frontend + Backend + PostgreSQL Database)

## Features
- Book Consulation Appointments
- PostgreSQL Database Integration
- Dockerized frontend and backend
- CI/CD Deployment to Azure VM via GitHub Actions

## How to run locally
1. Install Docker & Docker compose
2. Run 'docker compose up -d --build'
3. Access frontend at 'http://localhost:8080
